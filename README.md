# Sistema de Gestão - Python + MySQL

Sistema de linha de comando para gerenciamento de usuários e tarefas.

## Funcionalidades
- Cadastro e login de usuários
- CRUD de tarefas
- Relacionamento entre tabelas
- Logs de ações
- Variáveis de ambiente

## Tecnologias
- Python 3
- MySQL

## Como executar
1. Execute o arquivo sql/schema.sql no MySQL
2. Crie um arquivo .env baseado no .env.example
3. Instale as dependências:
   pip install -r requirements.txt
4. Execute:
   python main.py
